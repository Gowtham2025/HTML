What I do

Static Websites

A static web page is a web page that is delivered to the user's web browser exactly as stored, in contrast to dynamic web pages which are generated by a web application.

Responsive Web desing

Responsive web design is a web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it.

Customisable Layout

This composable allows you to measure and lay out children manually. All higher-level layouts like Column and Row are built with the Layout composable. Note: In the View system.
